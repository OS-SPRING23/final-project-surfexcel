How To Compile Virtual Machine Code and Run Assembly object

1: Download All Files From Repository
2: Save in one folder
3: Now go to that directory from Ubuntu Terminal
4: Use ls command to find name of file to compile
5: Now type gcc Virtual.c -o (whatevername)-vm
6: now again type ls to show vm compiled file
7: Now you can run whatever Assembly file you want to run
8: To Test simply type ./(whatevername)-vm (objectfilename).obj
9: It will run on a Virtual Machine.
